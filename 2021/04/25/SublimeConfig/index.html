sublime text设置中文的方法

1.安装package control

Ctrl+shift+p打开搜索栏

输入install查找，选择install package control

点击安装

2.安装中文插件

菜单栏preferences>package control

输入点击install package

再输入clz，点击ChineseLocalizations

安装完成

（需要换回英语或其他语言在Help>language中设置即可）
